# Diabetes Prediction Model

This project is a Machine Learning web application that predicts whether a person is diabetic or not using medical input data.

The application is built using:
- Python
- Scikit-learn
- Streamlit

---

# Features

- Predicts diabetes based on user input
- Simple and interactive Streamlit UI
- Machine Learning classification model
- Fast and lightweight application

---

# Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Streamlit

---

# Dataset Features

The model uses the following input features:

1. Pregnancies
2. Glucose Level
3. Blood Pressure
4. Skin Thickness
5. Insulin Level
6. BMI
7. Diabetes Pedigree Function
8. Age
